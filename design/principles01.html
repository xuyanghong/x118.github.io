    # 80/20 法则

    #

    #

    #

    # 定义：在所有大系统中，大部分的效果由少数几项关键因素决定。

    #

    80/20法则的使用范围非常普遍，包括遵循常态分配原则的系统，但是仅限于一些由许多细小琐碎、互不相关的事物影响的变数，例如，在同一系统中，因为人们的不同使用互补相同，应用案例如下：

    #

    • 产品80%的使用，只使用了产品的20%的功能

    • 城市80%的交通，集中在20%的道路上

    • 公司80%的收益， 归功于20%的产品

    • 80%的发明，归功于20%的人口

    • 80%的进步，来自于20%的努力

    • 80%的差错，来自于20%的零件

    #

    总结：有助于资源整合，可以提升设计的最大化。设计师对设计中的所有元素不能同等看待，需要利用该原则来评估设计过程中各要素的价值、划定中心设计和优化的范围，有效集中资源进行再设计。当时间和资源有限时，80%的非关键性功能，能减则减。
